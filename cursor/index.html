Cursor

Cursor हा AI-powered code editor आहे — VS Code वरच based आहे, त्यामुळे VS Code वापरत असाल तर एकदम familiar वाटेल. फरक एवढाच की यात AI built-in आहे. Code लिहायला, debug करायला, refactor करायला — सगळं AI ने fast होतं.

काय करता येतं?

  • AI Autocomplete — तुम्ही type करताना AI next lines suggest करतो. Tab press करा, code ready.
  • Inline Chat — कोणत्याही code वर select करा, Cmd+K / Ctrl+K press करा, AI ला सांगा काय change करायचं.
  • Full Chat Panel — sidebar मध्ये ChatGPT सारखं chat. तुमच्या codebase बद्दल questions विचारा, AI answer देतो.
  • Codebase-Aware — Cursor तुमचा पूर्ण project समजतो. एका file बद्दल विचारलं तरी related files पण बघतो.
  • Multi-file Editing — एका prompt मध्ये multiple files edit करता येतात. "हा button component बनव आणि page मध्ये add कर" — done.

कसं सुरू करायचं?

1

Download करा

cursor.com वर जा आणि तुमच्या OS साठी (Windows / Mac / Linux) download करा. Install करा — normal app सारखंच आहे.

2

Sign Up करा

Cursor उघडल्यावर sign up करा (Google / GitHub / Email). Free trial automatically start होतो.

3

Project open करा

File > Open Folder — तुमचा project folder select करा. VS Code वापरत असाल तर extensions आणि settings import करता येतात.

4

AI वापरायला सुरुवात करा

Cmd+K / Ctrl+K — inline edit. Cmd+L / Ctrl+L — chat panel. Type करताना autocomplete suggestions आपोआप येतील.

Try करा — Example Prompts

Chat मध्ये विचारा:

Explain this function in simple terms and suggest improvements

कोणताही function select करा, chat उघडा, आणि हे paste करा. AI function explain करेल आणि improvements सांगेल.

Inline edit (Cmd+K):

Add proper error handling and input validation to this function

Function select करा, Ctrl+K press करा, हे type करा. AI directly code edit करेल — Accept / Reject करा.

New file generate करा:

Create a responsive navbar component with dark mode toggle using React and Tailwind CSS

Chat मध्ये हे सांगा — Cursor पूर्ण component code लिहून देईल, file create करेल.

VS Code वापरता? Cursor चा interface VS Code सारखाच आहे — shortcuts, extensions, settings सगळं same. Migration 5 minutes मध्ये होतं. Import option sign-up वेळी येतं.
