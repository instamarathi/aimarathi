Prompting शिका

AI ला नक्की काय हवं ते सांगायला शिका — एक simple formula वापरून.

Objective

या module नंतर तुम्ही clear, specific prompts लिहू शकाल जे consistently चांगले results देतात.

Prompting Formula: TCCF

चांगला prompt लिहायला 4 गोष्टी लागतात:

T

Task — काय करायचं आहे?

AI ला clearly सांगा — explain करायचं आहे, summarize करायचं आहे, compare करायचं आहे, quiz बनवायचा आहे? Specific verb वापरा.

C

Context — background काय आहे?

तुम्ही कोण आहात, कोणत्या level ला आहात, कशासाठी हवं आहे. "मी 12th Science student आहे" हे सांगितलं तर AI output पूर्ण बदलतो.

C

Constraints — काय नको, किती हवं?

Word limit, language, difficulty level, काय include करू नये — या सगळ्या constraints आहेत. "200 words मध्ये", "technical terms avoid कर", "Hindi वापरू नकोस" — अशा गोष्टी.

F

Format — output कसा दिसायला हवा?

Bullet points, paragraph, table, quiz, step-by-step — format सांगितलं तर AI exactly त्या format मध्ये देतो.

Demo: Biology Exam Question

❌ वाईट prompt:

cell division explain कर

हा prompt खूप vague आहे. AI ला माहीत नाही — mitosis का meiosis? कोणत्या level साठी? किती detail हवं? Output generic येतो.

✅ चांगला prompt:

मी 12th Science (Biology) चा student आहे. मला mitosis आणि meiosis मधला difference explain कर.

Format असं हवं:
- पहिल्यांदा दोन्ही separately 3-4 lines मध्ये explain कर
- मग एक comparison table दे (stages, chromosome number, purpose)
- शेवटी 5 MCQs दे board exam pattern मध्ये, answers सोबत

Simple language वापर, पण scientific terms accurately वापर.

बघा — Task (explain + compare + quiz), Context (12th Science), Constraints (simple language, accurate terms), Format (explanation + table + MCQs) — सगळं clear आहे.

Important: Iterate करा

First prompt चा output perfect नसतो — आणि तो perfect असायला नको. AI शी conversation करा:

  • "हे खूप complex आहे, अजून simple कर"
  • "table format मध्ये दे"
  • "आता याच topic वर 5 short answer questions दे"
  • "हे answer चुकीचं वाटतंय, verify कर"

2-3 follow-ups नंतर output significantly better होतो. हा iterative approach शिकणं — हे prompting मधलं सगळ्यात important skill आहे.

Exercises

Exercise 1: Vague vs Better Prompt

खालचे दोन्ही prompts Gemini मध्ये try करा आणि outputs compare करा:

Vague prompt:

Indian economy बद्दल सांग

Better prompt:

मी 12th Commerce student आहे. मला Indian economy चे 5 major sectors सांग, प्रत्येक sector चा GDP मधला approximate share सांग, आणि 2024 मध्ये कोणता sector सगळ्यात fast grow होतोय ते सांग. Bullet points मध्ये दे, 200 words मध्ये.

दोन्ही outputs side by side बघा. फरक लक्षात येईल.

Exercise 2: Bad Prompt Improve करा

हा bad prompt घ्या आणि TCCF formula वापरून improve करा:

Bad prompt:

Shakespeare बद्दल essay लिहा

Hint: कोणता play/sonnet? किती words? कोणत्या aspect बद्दल — language, themes, historical context? कोणत्या level साठी?

Exercise 3: Same Task, 3 Formats

एकच topic घ्या (e.g., "Water cycle") आणि 3 different formats मध्ये output मागा:

Water cycle explain कर — 10th std level, मराठी-English mix मध्ये.
Format 1: एक short summary (100 words)
Format 2: step-by-step bullet points
Format 3: 10 quiz questions with answers

तिन्ही separately try करा किंवा एकदम तिन्ही मागा — बघा AI कसं handle करतो.

Copy-Paste Prompts

हे ready-to-use prompts आहेत. [ ] मधला text तुमच्या details ने replace करा.

Topic Explanation

मला [topic] समजावून सांग. मी [10th / 12th / college] student आहे. Simple language मध्ये explain कर, examples दे, आणि शेवटी 5 quiz questions दे.

Paragraph Rewrite

हा paragraph rewrite कर — same meaning, पण clearer structure आणि simpler words. Paragraph: [paste your paragraph here]

Multiple Perspectives

या topic वर 3 different perspectives दे: [topic]. प्रत्येक perspective साठी — argument, evidence, weakness सांग.

Exam Preparation

[Subject] च्या [chapter name] वरचे 10 most important questions सांग — [board/university] exam pattern मध्ये. Short answer आणि long answer दोन्ही include कर, model answers सोबत दे.

Concept Comparison

[Concept A] आणि [Concept B] मधला difference explain कर. Table format मध्ये comparison दे, मग real-life example दे, शेवटी 3 tricky MCQs दे जे दोन्ही concepts test करतात.
Key Takeaway: Prompting ही skill आहे — practice ने improve होते. TCCF formula लक्षात ठेवा: Task + Context + Constraints + Format. आणि always iterate करा — first output हा starting point आहे, final answer नाही.
