Tips & Tricks

Claude Code मधून maximum काढायला या tips वापरा

Tip 1: CLAUDE.md file वापरा

Project root मध्ये CLAUDE.md नावाची file बनवा. त्यात project बद्दल instructions लिहा — Claude प्रत्येक वेळी हे file वाचतो session सुरू करताना.

Example:

CLAUDE.mdThis is a React project. Use TypeScript. Follow existing code style. Always write tests for new features.

हे म्हणजे Claude ला एकदाच सांगा, तो लक्षात ठेवतो. पुन्हा पुन्हा same instructions द्यायची गरज नाही!

Tip 2: Specific prompts लिहा

Claude ला जेवढं specific सांगाल, तेवढं चांगलं output मिळेल. Bad vs Good prompts बघा:

Bad promptFix this

Claude ला काय fix करायचं ते कळत नाही — कोणता error? कोणती file?

Good promptFix the login error — when I click submit, it shows 'undefined is not a function' in console
Bad promptMake it better

काय better? Speed? Design? Readability? Claude ला guess करायला लागतं.

Good promptMake the homepage load faster by lazy-loading images

Tip 3: /compact वापरा

लांब conversation मध्ये Claude चा context window भरतो — response slow होतात आणि accuracy कमी होते.

Claude Code prompt/compact

/compact ने conversation compress होते — Claude faster आणि accurately respond करतो. जुने messages summarize होतात, पण important context टिकतो.

Rule of thumb

20+ messages झाले की /compact करा. Conversation खूप लांब गेली असं वाटलं तर लगेच करा.

Tip 4: Git integration

Claude Code git बरोबर deeply integrate आहे. तुम्ही git commands manually लिहायची गरज नाही — natural language मध्ये सांगा:

Claude Code promptCommit my changes with a good message

Claude descriptive commit message लिहितो — changed files बघून automatic.

Claude Code promptCreate a new branch called feature-login and switch to it
Claude Code promptShow me what changed since yesterday

Git init नसेल तर?

Project मध्ये git init नसेल तर Claude ला सांगा: "Initialize git in this project" — तो setup करेल.

Tip 5: Multiple files एकदम handle करा

Claude ला whole project context असतो — एक file नाही, सगळ्या files. म्हणून batch operations सहज होतात:

Claude Code promptRename all .js files to .ts and update imports

Claude सगळ्या files बदलतो — imports, references सगळं update करतो.

Claude Code promptAdd error handling to all API calls in the src folder

एकेक file उघडून बदलायची गरज नाही — Claude एकदम सगळं करतो.

Tip 6: Learn from Claude

Claude Code हा फक्त code लिहायचा tool नाही — हे coding शिकायचा fastest way आहे. Personalized tutor!

Claude Code promptExplain what you just wrote, line by line

Code लिहिल्यावर हे विचारा — प्रत्येक line काय करते ते समजेल.

Claude Code promptWhy did you choose this approach instead of using a for loop?
Claude Code promptWhat are the potential bugs in this code?

Claude तुम्हाला code review पण करून देतो — bugs, edge cases, improvements सगळं सांगतो.

Tip 7: Cost-saving tips

  • Free tier limited आहे — मोठे projects साठी Pro plan ($20/month) किंवा Max plan ($100/month) लागतो
  • /compact वापरा — कमी tokens = कमी cost
  • Clear, specific prompts = कमी back-and-forth = कमी usage
  • छोटे tasks free tier मध्ये सहज होतात — experiments करा

Tip 8: Common mistakes avoid करा

  • Sensitive information (passwords, API keys) Claude ला देऊ नका — ते Anthropic च्या servers वर जातं
  • Production server वर directly Claude Code run करू नका — development environment वापरा
  • Claude चं सगळं output blindly trust करू नका — review करा, test करा
  • Important files चा backup ठेवा — changes करण्याआधी git commit करा

Summary — Quick Checklist

  • CLAUDE.md file बनवा — project instructions एकदाच लिहा
  • Specific prompts लिहा — "Fix this" ऐवजी detail द्या
  • /compact वापरा — लांब conversations compress करा
  • Git वापरा — Claude ला natural language मध्ये git commands सांगा
  • Claude कडून शिका — "Explain this" विचारा
  • Cost manage करा — specific prompts + /compact = कमी tokens
  • Safe राहा — passwords देऊ नका, output review करा, backup ठेवा
