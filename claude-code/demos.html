Claude Code Demos

Real examples — Claude Code ने काय काय करता येतं ते बघा

Demo 1: Portfolio Website बनवा

30 minutes मध्ये

Scenario: तुम्हाला एक personal portfolio website बनवायची आहे — About Me, Skills, Projects, Contact सगळं असलेली. पण तुम्हाला HTML/CSS येत नाही. No problem!

Step by step prompts:

1

Website structure सांगा

PromptCreate a personal portfolio website. My name is [तुमचं नाव]. I'm a student from Maharashtra. Include sections: About Me, Skills, Projects, Contact. Use a modern dark theme.

Claude HTML, CSS, JS files बनवेल — तुम्हाला approve करायला सांगेल. Yes म्हणा.

2

Project card add करा

PromptAdd a project card for my college assignment — a calculator app built with Python
3

Mobile-friendly बनवा

PromptMake it mobile-responsive
4

Contact form add करा

PromptAdd a contact form that opens mailto
Result: 30 minutes मध्ये complete portfolio ready! बिना एक line code लिहिता.
Tip: हे portfolio directly GitHub Pages वर host करता येतं — free!

Demo 2: Python Script लिहा

10 minutes मध्ये

Scenario: College assignment आहे — CSV file मधला data analyze करायचा आहे. Python script लिहायला सांगा, Claude लिहून देईल!

Step by step prompts:

1

Analysis script बनवा

PromptCreate a Python script that reads a CSV file called 'marks.csv' with columns: name, subject, marks. Calculate average marks per student and per subject. Show results in a nice table.
2

Sample data बनवा

PromptAlso create a sample marks.csv with 10 students and 5 subjects for testing
3

Chart add करा

PromptAdd a bar chart showing average marks per subject using matplotlib
4

Script run करा

Promptuv run python analyze.py

Claude script run करेल आणि output दाखवेल — table आणि chart दोन्ही!

Result: Complete data analysis script + chart — assignment ready!

Demo 3: Content लिहा

15 minutes मध्ये

Scenario: College fest साठी event description, social media posts, आणि WhatsApp forward message लिहायचं आहे. सगळं Claude ला सांगा!

Step by step prompts:

1

सगळं content एकदम सांगा

PromptWrite content for our college tech fest 'TechVista 2026'. Write: 1) A formal event description (200 words), 2) 5 Instagram caption ideas in Marathi-English mix, 3) A WhatsApp forward message in Marathi
2

Instagram captions improve करा

PromptMake the Instagram captions more fun and add relevant hashtags
3

Poster बनवा

PromptCreate an HTML poster layout for the event with date April 25, 2026, venue: College Auditorium
Result: Complete marketing content ready to use — event description, Instagram posts, WhatsApp message, आणि poster!

Demo 4: Existing Code समजून घ्या

5 minutes मध्ये

Scenario: GitHub वरून code download केला पण काही समजत नाही. कोणत्या file मध्ये काय आहे, main function काय करतो — काहीच माहीत नाही. Claude ला विचारा!

Step by step prompts:

1

Project समजून घ्या

PromptRead all the files in this project and explain what this project does, in simple Marathi
2

Main function explain करा

PromptExplain the main function step by step
3

नवीन feature कसं add करायचं विचारा

PromptWhat would I need to change to add a new feature that does [X]?
Result: कोणताही codebase 5 minutes मध्ये समजतो — मराठीत explanation!
