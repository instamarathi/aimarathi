Claude Code Windows वर Install करा

Step-by-step guide — प्रत्येक step सोपा आहे, follow करा

Prerequisites — आधी हे check करा

  • Windows 10 version 1903+ किंवा Windows 11 लागेल
  • Internet connection लागेल
  • Admin access लागेल (install करायला)
1

Node.js Install करा

Claude Code ला Node.js लागतं — हे एक runtime आहे जे JavaScript चालवतं. Claude Code हे Node.js वर बनलंय, म्हणून आधी हे install करायला लागेल.

  • nodejs.org वर जा → "LTS" version download करा
  • Downloaded file वर double-click करा → Next → Next → Next → Install
  • Installation मध्ये "Add to PATH" checkbox tick आहे ना ते बघा
📸 Node.js installer — LTS version select करा

Check करा: Command Prompt उघडा, type करा:

Command Promptnode --version

Version number दिसला (जसं v20.x.x) म्हणजे Node.js नीट install झालंय!

2

Anthropic Account बनवा

Claude Code हे Anthropic ने बनवलंय. वापरायला त्यांचं account लागतं.

  • console.anthropic.com वर जा
  • Sign up करा — Google account ने पण होतं
  • Free tier मध्ये पण Claude Code वापरता येतो
  • Pro plan ($20/month) मध्ये जास्त usage मिळतो — पण सुरुवातीला free पुरेसं आहे

Plan बद्दल Note

Claude Code ला Max plan ($100/month) किंवा API key लागतो for unlimited use. सुरुवातीला free tier ने experiment करा, आवडलं तर upgrade करा.

3

Claude Code Install करा

आता actual Claude Code install करायचं — एक command आहे, बस!

  • Command Prompt (cmd) उघडा — Start menu मध्ये cmd search करा
  • खालची command type करा आणि Enter दाबा:
Command Promptnpm install -g @anthropic-ai/claude-code
  • Wait करा — download होईल (1-2 minutes लागतील)
  • बऱ्याच lines येतील screen वर — घाबरू नका, ते normal आहे
📸 npm install command running

Done message आलं की installation complete!

4

पहिल्यांदा Run करा

Install झालं — आता Claude Code चालवू या!

  • Command Prompt मध्ये type करा:
Command Promptclaude
  • पहिल्यांदा Anthropic login मागेल — browser automatically उघडेल
  • Browser मध्ये login करा (Step 2 मध्ये बनवलेलं account वापरा)
  • Login झालं की terminal मध्ये Claude ready दिसेल
📸 Claude Code first run — login prompt
5

Test करा

सगळं नीट काम करतंय का ते बघू या:

Claude Code promptHello! Tell me about yourself in Marathi

Claude ने Marathi मध्ये reply दिला? Congratulations — सगळं काम करतंय! तुम्ही successfully Claude Code install केलंय.

Troubleshooting — काही problem आला?

"node is not recognized" error आला?

Node.js install नीट झालं नाही. Reinstall करा आणि installation मध्ये "Add to PATH" checkbox tick करा. Reinstall केल्यावर Command Prompt बंद करून नवीन उघडा.

"npm ERR!" error आला?

Internet connection check करा. जर college/office मध्ये proxy मागे असाल तर IT admin ला विचारा — proxy settings set करायला लागतील.

Login page उघडत नाही?

Browser default check करा. काम नाही झालं तर terminal मध्ये दिसणारी URL manually copy करा आणि browser मध्ये paste करा.
