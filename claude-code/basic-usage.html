Claude Code — Basic Usage Guide

Install झालं? आता शिका कसं वापरायचं

1. Claude Code कसं start करायचं

Claude Code हे terminal (Command Prompt / PowerShell / Mac Terminal) मध्ये चालतं. Start करणं खूप simple आहे:

1

तुमच्या project folder मध्ये जा

Command Prompt उघडा आणि cd command वापरून तुमच्या project folder मध्ये जा:

Terminalcd path\to\your\folder

Example: cd Desktop\my-website

2

claude type करा

Terminal मध्ये फक्त claude type करा आणि Enter दाबा — Claude start होईल!

Terminalclaude
Tip: जिथे तुमचा project आहे तिथे cd करा — Claude ला तुमच्या files चा context मिळतो. म्हणजे Claude ला कळतं तुम्ही कशावर काम करत आहात.

2. Claude शी कसं बोलायचं

Claude ला natural language मध्ये सांगा काय करायचं — English किंवा Marathi दोन्ही चालतं. एखाद्या माणसाला सांगतो तसं सांगा.

Specific असा! "make a website" पेक्षा खालील prompt जास्त चांगलं काम करतो:

Example promptMake a portfolio website with dark theme and 3 sections: about, projects, contact

आणखी काही examples:

PromptCreate a new file called index.html with a basic webpage
PromptRead the file app.py and explain what it does
PromptFix the error in server.js — it shows 'cannot read property of undefined'
PromptAdd a contact form to my website

3. Claude चं response कसं समजून घ्यायचं

Claude code लिहितो तेव्हा तो diff format मध्ये दाखवतो:

  • हिरवा (green) = नवीन code added
  • लाल (red) = जुना code removed

Permission system:

  • Claude file edit करायच्या आधी permission मागतो — Y (yes) किंवा N (no) press करा
  • Claude terminal commands run करायच्या आधी पण permission मागतो — command वाचा, मग approve करा
Tip: Permission prompt वर नीट वाचा काय होतंय — especially जेव्हा Claude file delete किंवा system command run करायला सांगतो.

4. Key Commands

Claude Code मध्ये काही important commands आहेत:

Command काय करतो
/help Help menu दाखवतो
/clear Conversation reset करा (नवीन topic साठी)
/compact Conversation compress करा (लांब session मध्ये useful)
Ctrl+C Current generation stop करा
exit किंवा Ctrl+D Claude Code बंद करा

5. Beginners साठी Tips

Tip: एकदम perfect prompt लिहायची गरज नाही — Claude ला follow-up करता येतो. आधी basic सांगा, मग refine करा.
Tip: चुकलं तर घाबरू नका — Claude undo करू शकतो, आणि git revert आहेच backup म्हणून.
Tip: मोठं काम असेल तर छोट्या steps मध्ये सांगा. एकदम "build me a full app" पेक्षा step by step approach चांगला.

Try it yourself!

एक नवीन folder बनवा, त्यात Claude start करा, आणि सांगा:

PromptCreate a simple HTML page that says Hello World with a nice design

बघा काय बनतं! Claude ने बनवलेली file browser मध्ये उघडा आणि enjoy करा.
