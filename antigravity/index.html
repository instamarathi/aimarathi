Google Antigravity

Google चा AI-powered IDE. Agent-first coding — AI agents तुमच्यासाठी complex code लिहितात. Cursor/Windsurf सारखंच, पण Google चं — आणि सध्या FREE आहे.

काय करता येतं?

  • Agent-first coding — AI agents autonomously code लिहितात, files edit करतात. तुम्ही सांगा काय बनवायचं, agent करतो.
  • Editor + Manager view — VS Code सारखं editor view, plus multiple agents एकत्र manage करण्यासाठी manager view. एकाच वेळी अनेक tasks चालवा.
  • Multiple AI models — Gemini 3.1 Pro, Gemini Flash, Claude Sonnet, Claude Opus — तुम्हाला हवं ते model choose करा.
  • Artifacts — task lists, implementation plans — verifiable deliverables मिळतात, not just raw output. Agent काय करतोय ते clearly दिसतं.
  • Integrated browser + terminal — agents ला directly browser आणि terminal access आहे. Website बनवली तर लगेच preview दिसतो.

किती खर्च येतो?

PlanPriceकाय मिळतं
Public Preview$0 (FREE)सगळे features वापरता येतात. Gemini models साठी generous rate limits. Claude models साठी काही limits असू शकतात.
Tip: Future pricing अजून announce झालेलं नाही. सध्या free आहे तोपर्यंत जास्तीत जास्त वापरा — Cursor/Windsurf चा $15-20/month paid plan घेण्यापूर्वी Antigravity try करा.

कसं सुरू करायचं?

1

Website वर जा

antigravity.google वर जा. Install काहीच करायला नको — browser मध्येच चालतं.

2

Google account ने sign in करा

तुमचं Google account वापरून sign in करा. Gmail account असेल तर तेच चालतं.

3

Project open करा किंवा नवीन बनवा

Existing project import करा (GitHub वरून) किंवा blank project सुरू करा.

4

Agent ला सांगा काय बनवायचं

Chat मध्ये तुमची idea describe करा — agent code लिहितो, files बनवतो, test करतो. Manager view मध्ये multiple agents ला parallel tasks द्या.

Try करा — Example Prompts

Website बनवा:

Build a modern portfolio website with dark theme, hero section, projects gallery, and contact form. Make it mobile responsive with smooth scroll animations.

Agent पूर्ण website बनवतो — HTML, CSS, JS सगळं. Built-in browser मध्ये लगेच preview दिसतो.

Full-stack app बनवा:

Create a task management app with user authentication, SQLite database, and a clean dashboard. Users should be able to create projects, add tasks, set deadlines, and track progress.

Agent backend + frontend दोन्ही बनवतो. Database setup, API routes, UI — सगळं handle करतो.

Debug करा:

This app crashes when I click the submit button. Check the form handling code, find the bug, and fix it. Also add proper error handling.

Agent code वाचतो, bug शोधतो, fix करतो, आणि test करतो — सगळं automatically.

सध्या free आहे — वापरा! Google चे Gemini models unlimited आहेत preview मध्ये. Cursor/Windsurf चा $15-20/month paid plan घेण्यापूर्वी Antigravity try करा. Browser-based आहे त्यामुळे install शिवाय लगेच सुरू करता येतं.
