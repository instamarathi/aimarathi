Perplexity

AI-powered search engine — Google ला AI चा alternative. तुम्ही प्रश्न विचारता, Perplexity internet search करतो, multiple sources वाचतो, आणि एक clear answer देतो — प्रत्येक fact साठी source link दाखवतो.

Google वर search करताना 10 links उघडून वाचायला लागतात. Perplexity हे सगळं तुमच्यासाठी करतो आणि एक summarized answer देतो. Research, fact-checking, current events — या सगळ्यासाठी Perplexity best आहे.

Research साठी सगळ्यात reliable: ChatGPT आणि Gemini कधी कधी चुकीची information देतात शिवाय source न सांगता. Perplexity प्रत्येक sentence ला source link देतो — त्यामुळे verify करणं सोपं होतं.

Perplexity ने काय काय करता येतं?

  • Cited Answers: प्रत्येक answer ला numbered source citations — कुठून information आली ते exactly कळतं
  • Real-time Search: Latest news, stock prices, weather — real-time data instantly मिळतो
  • Deep Research: "Focus" mode मध्ये academic papers, Reddit, YouTube — specific sources मधून search करता येतो
  • Follow-up Questions: एका topic वर deep dive करा — Perplexity context लक्षात ठेवतो आणि related questions suggest करतो
  • File & Image Upload: Documents upload करून त्यावर questions विचारा, images upload करून information मिळवा

कसं सुरू करायचं?

  1. perplexity.ai उघडा — account शिवाय पण search करता येतो
  2. Sign up करा (Google account ने) — search history save होतो, collections बनवता येतात
  3. Search box मध्ये तुमचा प्रश्न natural language मध्ये type करा
  4. "Focus" button वापरून specific sources निवडा — Academic, Reddit, YouTube, etc.
