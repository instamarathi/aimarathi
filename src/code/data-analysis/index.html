Data Analysis

CSV, Excel, JSON, databases — कोणत्याही data द्या, AI analyze करतो. Charts बनवतो, patterns शोधतो, insights काढतो. College assignments साठी पण उपयोगी, business साठी पण.

कोणते Tools वापरायचे?

Tool Free / Paid Notes
Gemini Free Google चा AI. File upload करा (CSV, Excel), direct analysis मिळतो. Charts पण बनवतो. Free आहे — सुरुवात इथून करा.
ChatGPT (Code Interpreter) Paid ($20/mo) File upload करा, Python code run करतो, charts बनवतो. Advanced analysis साठी best.

Recommend: Gemini ने सुरुवात करा

Gemini free आहे आणि file upload support आहे. CSV किंवा Excel upload करा, question विचारा — charts आणि insights मिळतात. पैसे खर्च शिवाय data analysis शिकता येतो.

Copy-Paste Prompts

तुमची file upload करा आणि हे prompts paste करा. File name तुमच्या file नुसार बदला.

Sales Data Analysis

Analyze this sales data. Show me: 1) Total revenue and average order value, 2) Month-wise sales trend as a line chart, 3) Top 5 best-selling products, 4) Which day of the week has most sales, 5) Category-wise revenue breakdown as a pie chart. Highlight any interesting patterns or anomalies.

Survey Results Analysis

Analyze this survey data. Show me: 1) Response summary for each question with percentages, 2) Bar charts for multiple-choice questions, 3) Common themes in open-ended responses, 4) Demographic breakdown of respondents, 5) Cross-tabulation between age group and satisfaction rating. Summarize key findings in 5 bullet points.

Monthly Expense Tracking

Analyze my expense data. Show me: 1) Total spending per month as a bar chart, 2) Category-wise breakdown (Food, Transport, Bills, Entertainment, Shopping) as a pie chart, 3) Average daily spending, 4) Which category am I overspending on compared to previous months, 5) Predict next month's expenses based on the trend. Give me 3 practical tips to save money.

Student Marks Analysis (College Projects साठी)

Analyze this student marks data. Calculate: 1) Class average, median, highest, and lowest for each subject, 2) Subject-wise pass/fail percentage, 3) Grade distribution (A/B/C/D/F) as a bar chart, 4) Correlation between subjects — which subjects' marks move together, 5) Top 10 students by total marks. Create a summary report suitable for a college presentation.

Pro Tips

  • File upload करण्यापूर्वी data clean असलं तर चांगलं — column names English मध्ये, empty rows काढा.
  • "Explain this like I'm a beginner" add करा prompt मध्ये — AI simple भाषेत सांगतो.
  • Chart आवडलं नाही? "Make it a bar chart instead" किंवा "use blue color scheme" सांगा.
  • AI ने दिलेला Python code save करा — पुन्हा same analysis करायची असेल तर code run करा.

लक्षात ठेवा

Sensitive data (salary, personal info) upload करताना काळजी घ्या. AI tools चे servers बाहेरच्या देशात आहेत. Confidential business data साठी company policy check करा. Practice साठी sample/dummy data वापरा.
