GitHub Copilot

GitHub Copilot हा GitHub चा official AI coding assistant आहे. तुमच्या editor मध्ये directly बसतो — VS Code, JetBrains, Neovim — आणि real-time code suggestions देतो. GitHub ecosystem शी tight integration असल्यामुळे, repo context चांगला समजतो.

काय करता येतं?

  • Autocomplete (Ghost Text) — तुम्ही type करताना पुढच्या lines gray text मध्ये दिसतात. Tab press करा — done. Industry मधला best autocomplete experience आहे.
  • Copilot Chat — VS Code sidebar मध्ये AI chat. Code explain करा, bugs शोधा, tests generate करा — सगळं chat मधून.
  • Inline Chat — Ctrl+I press करा, directly code मध्ये AI ला instruction द्या. Select केलेला code edit करतो.
  • GitHub Integration — Pull request summaries, code review suggestions, commit message generation. GitHub वर काम करत असाल तर हे seamless आहे.
  • Multi-language Support — Python, JavaScript, TypeScript, Go, Java, C#, Ruby — practically सगळ्या languages support करतो.

किती खर्च येतो?

PlanPriceकाय मिळतं
Free$0Limited completions आणि chat messages per month. Open source contributors आणि students ना free मिळतं.
Individual$10/monthUnlimited completions, unlimited chat, सगळे editors support
Business$19/user/monthOrganization-wide policies, IP indemnity, admin controls
Tip: Student असाल तर GitHub Student Developer Pack apply करा — Copilot Individual free मिळतं. .edu email लागतो. Student नसाल तर एक महिना $10 चा Individual plan try करा — Copilot ने save होणारा coding time बघता subscription लगेच वसूल होतो.

कसं सुरू करायचं?

1

GitHub account बनवा

github.com वर account नसेल तर बनवा. Copilot हे GitHub चं product आहे, त्यामुळे GitHub account हवंच.

2

Copilot activate करा

GitHub Settings > Copilot वर जा. Free plan select करा किंवा Individual plan subscribe करा. Student pack असेल तर automatically available होतं.

3

VS Code extension install करा

VS Code मध्ये Extensions tab उघडा, "GitHub Copilot" search करा, Install करा. GitHub login करा — extension automatically activate होईल.

4

Code लिहायला सुरुवात करा

कोणतीही file उघडा, type करायला सुरुवात करा — gray text मध्ये suggestions दिसतील. Tab — accept. Esc — dismiss. Ctrl+I — inline chat.

Try करा — Example Prompts

Comment लिहा, code मिळवा:

// function to validate email address and return true/false

हा comment लिहा आणि Enter press करा. Copilot पूर्ण function suggest करेल — Tab press करा.

Copilot Chat मध्ये विचारा:

/explain what does this code do and are there any potential bugs?

Code select करा, Ctrl+I press करा, हे type करा. Copilot code explain करेल आणि bugs point out करेल.

Tests generate करा:

/tests generate unit tests for this function with edge cases

Function select करा, Copilot Chat मध्ये हे type करा. Tests automatically generate होतील — file मध्ये paste करा.

Cursor vs Copilot? Copilot चा autocomplete experience सगळ्यात smooth आहे — tab-tab-tab करत code लिहिता येतो. पण Cursor चा chat आणि multi-file editing powerful आहे. दोन्ही try करा, तुम्हाला जे जास्त suit होतं ते वापरा.
