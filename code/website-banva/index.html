Website बनवा

Portfolio, blog, business site — AI ला सांगा काय बनवायचं, तो HTML, CSS, JS लिहितो. एक line code न लिहिता.

कोणते Tools वापरायचे?

Tool Free / Paid Notes
Claude Code Paid ($20/mo) Terminal-based, best quality output. तुम्ही describe करा, तो full website लिहितो. सगळ्यात reliable.
Cursor Free tier + Paid VS Code-based editor. Code दिसतो, real-time edit करता येतो. Visual learners साठी चांगलं.
Windsurf Free tier + Paid Cursor सारखंच, VS Code-based. AI suggestions चांगल्या येतात.
Antigravity Free (preview) Google चा AI IDE — agent-first coding, Gemini + Claude models. Browser-based, install नको.
Bolt.new Free tier available Browser-based — install काहीच नाही. Link share करा, website ready. Beginners साठी best start.

कुठून सुरुवात करायची?

अजिबात coding माहीत नसेल तर Bolt.new वापरा — browser मध्येच सगळं होतं. थोडं comfortable झालात की Claude Code किंवा Cursor try करा. Antigravity सध्या free आहे आणि browser-based आहे — paid tool घेण्यापूर्वी नक्की try करा. Paid versions मध्ये output quality आणि speed मध्ये खूप फरक पडतो — मित्रांसोबत subscription share करा, per person cost कमी होतो.

Copy-Paste Prompts

हे prompts direct tool मध्ये paste करा. तुमच्या गरजेनुसार details बदला.

Portfolio Website

Build a personal portfolio website with these sections: Hero with my name "तुमचं नाव" and tagline "तुमचं designation", About Me, Skills (Python, JavaScript, React), Projects (3 cards with image placeholders, title, description, and GitHub link), and Contact form. Use modern dark theme, smooth scroll, mobile responsive. Add subtle animations on scroll.

Restaurant Menu Site

Create a restaurant website for "तुमच्या restaurant चं नाव". Include: hero image section, menu page with categories (Starters, Main Course, Desserts, Drinks) with prices in ₹, photo gallery, location with embedded Google Maps placeholder, contact info and opening hours. Design should feel warm and inviting, mobile-friendly.

NGO / संस्था Landing Page

Build a landing page for an NGO called "तुमच्या संस्थेचं नाव" working on education in rural Maharashtra. Sections: Hero with mission statement, Impact stats (students helped, villages reached, schools built), Our Programs (3 cards), Team section, Donate button (link to #), testimonials, and footer with contact. Use hopeful, clean design with green/white colors.

Small Business Website

Create a website for a local business "तुमच्या business चं नाव" that sells homemade pickles (लोणची). Include: homepage with hero banner, product catalog with 6 items (name, price in ₹, weight, description), customer reviews section, WhatsApp order button (link to wa.me/तुमचा-number), About Us story, and delivery information. Bright, appetizing design.

Pro Tips

  • Prompt मध्ये specific details द्या — colors, sections, content. जितकं specific, तितकं चांगलं output.
  • पहिल्यांदा basic version बनवा, मग "add animations" किंवा "make it mobile responsive" असं सांगा.
  • AI ने बनवलेलं आवडलं नाही? "Change the color scheme to blue and white" असं सांगा — पुन्हा सुरुवातीपासून सांगायची गरज नाही.
  • Website deploy करायची असेल तर Netlify किंवा GitHub Pages वापरा — दोन्ही free आहेत.

लक्षात ठेवा

AI generated website हा starting point आहे. Production साठी वापरायचं असेल तर contact form backend, proper hosting, आणि domain name लागेल. पण prototype किंवा personal site साठी AI output directly वापरता येतो.
