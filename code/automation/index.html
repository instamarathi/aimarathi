Automation & Scripts

Repetitive काम automate करा — file renaming, data cleanup, email automation, cron jobs. AI ला सांगा काय करायचं, तो Python/Bash script लिहितो. रोज 2 तास वाचवा.

कोणते Tools वापरायचे?

Tool Free / Paid Notes
Claude Code Paid ($20/mo) Terminal-based — script लिहितो, test करतो, fix करतो. File system access असल्यामुळे automation साठी best.
ChatGPT Free tier + Paid Script लिहून देतो, explain करतो. Free tier मध्ये पण चांगलं काम करतो.
Gemini Free Google चा AI — free आहे. Script लिहून देतो, Google Workspace integration साठी especially useful.

Recommend: Gemini ने सुरुवात करा

Gemini free आहे — simple scripts साठी चांगलं काम करतो. Complex automation लागलं (multiple files, error handling) तर Claude Code वापरा — तो directly तुमच्या system वर script run करतो.

Copy-Paste Prompts

हे prompts direct tool मध्ये paste करा. File paths आणि details तुमच्या गरजेनुसार बदला.

File Renaming Script

Write a Python script that renames all files in a folder. Rules: 1) Convert filenames to lowercase, 2) Replace spaces with hyphens, 3) Add today's date as prefix (YYYY-MM-DD), 4) Handle duplicate names by adding -1, -2 suffix. Example: "My Report.pdf" becomes "2024-01-15-my-report.pdf". Add a dry-run mode that shows what will change without actually renaming. Print a summary of changes.

Email Automation Script

Write a Python script that reads a CSV file with columns (Name, Email, Amount) and sends personalized payment reminder emails. Use SMTP. Email template: "Hi {Name}, this is a friendly reminder that ₹{Amount} is pending. Please pay by end of this week." Add: 1) Log which emails were sent, 2) Delay between emails to avoid spam filters, 3) Error handling for invalid emails. Include setup instructions for Gmail SMTP.

Data Cleanup Script

Write a Python script to clean messy CSV data. Tasks: 1) Remove duplicate rows, 2) Fix phone numbers — ensure all are 10-digit format (remove +91, spaces, dashes), 3) Standardize date formats to YYYY-MM-DD, 4) Fill empty cells in "City" column with "Unknown", 5) Remove rows where email column is empty, 6) Save cleaned data as new CSV. Show before/after row count and summary of changes.

Folder Organizer Script

Write a Python script that organizes files in my Downloads folder into subfolders by type. Rules: Images (.jpg, .png, .gif, .svg) go to "Images/", Documents (.pdf, .doc, .docx, .xlsx) to "Documents/", Videos (.mp4, .mkv, .avi) to "Videos/", Code (.py, .js, .html, .css) to "Code/", everything else to "Others/". Create folders if they don't exist. Show summary: how many files moved to each folder.

Pro Tips

  • Script run करण्यापूर्वी dry-run mode मागा — आधी काय होणार ते बघा, मग actually run करा.
  • "Add error handling" सांगायला विसरू नका — real data messy असतो, script crash होता कामा नये.
  • AI ने दिलेला script आधी test data वर run करा, production data वर नाही.
  • Recurring task असेल तर "make it a cron job" किंवा "add scheduling" सांगा — script automatically run होईल.

लक्षात ठेवा

Automation scripts तुमच्या files, emails, data ला touch करतात. कोणताही script run करण्यापूर्वी backup घ्या. Dry-run आधी करा. Password किंवा API keys script मध्ये hardcode करू नका — environment variables वापरा.
